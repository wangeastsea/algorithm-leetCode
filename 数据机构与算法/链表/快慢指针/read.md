> 真题描述：给定一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

> 示例： 给定一个链表: 1->2->3->4->5, 和 n = 2.
当删除了倒数第二个结点后，链表变为 1->2->3->5.
说明： 给定的 n 保证是有效的。


let nodeList = {
    val: 1,
    next: {
        val: 2,
        next: {
            val: 3,
            next: {
                val: 4, 
                next: {
                    val: 5
                    next: null
                }
            }
        }
    }
}
